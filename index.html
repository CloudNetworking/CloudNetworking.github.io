Struts2简介

Apache Struts 2是一个用于开发Java EE网络应用程序的开源Web应用框架。它利用并扩展了Java Servlet API 注意：它只是表现层（MVC）框架。 Struts2 是一个非常优秀的MVC框架，基于Model2 设计模型，由传统Struts1和WebWork两个经典框架发展而来。

Action的动态方法调用

2017-02-09
Struts2访问Action
aa

通过url动态指定调用Action哪个方法而无需配置的method属性

开启动态调用:

<constant name="struts.enable.DynamicMethodInvocation" value="true"></constant>

通过 i方法名 指定调用Action哪个方法

localhost:8080/struts01/user!add.action

struts.xml没有指定method属性

但是 user!add.action 就会执行 UserAction 的 add 方法
